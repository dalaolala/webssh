# 功能特性

## ✅ 已实现功能

### 用户认证系统
- 用户注册（邮箱+密码）
- 用户登录（JWT令牌认证）
- 密码加密存储（bcrypt）
- 路由守卫和权限控制

### 服务器资产管理
- 服务器增删改查
- 支持密码和私钥认证
- AES-256加密存储敏感信息
- 服务器分组管理
- 树形结构展示

### SSH终端核心功能
- 基于xterm.js的Web终端
- WebSocket实时通信
- SSH2库后端连接
- 终端窗口自适应
- 命令输入和输出
- 快捷键支持（Ctrl+C等）

### 前端界面
- Vue3 + Element Plus现代化界面
- 响应式设计
- 服务器管理面板
- 终端界面
- 快速连接功能
